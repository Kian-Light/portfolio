Current Focus

Project: Integrated Catchment Characterisation for Groundwater Vulnerability (R + QGIS)

Building a reproducible R-based workflow that automates watershed delineation and stream network extraction from raw DEM data across Cheshire and Greater Manchester, then extends into groundwater vulnerability and recharge potential using BGS geology data. Currently in the scoping and setup phase: sourcing DEM tiles and establishing a consistent coordinate reference system before any automation work begins.

Timeline Roadmap

  • Phase 1 — Completed

    Project Scoping Done

  • Phase 2 — In Progress

    Data Sourcing & Terrain Pre-Processing In Progress

    Sourcing DEM tiles for the Cheshire / Greater Manchester study area and reprojecting to a consistent metric CRS (EPSG:27700). Prototyping r.watershed flow accumulation thresholds on a single test basin in QGIS before scripting the loop.

    • → Sourced raw DEM raster tiles from national data portal (Current Task)
    • ○ Reprojected spatial data frames to standard metric CRS (EPSG:27700)
    • ○ Prototyping r.watershed flow accumulation parameters in QGIS
    • ○ Scripting the global batch processing loop array
  • Phase 3 — Next Step

    Automation via qgisprocess Planned

    Drafting the core R loop array with the qgisprocess package. The loop pipeline will automatically pipe the processed raster folders into r.watershed routines, calculating continuous hydrographic basins and generating vectorized stream networks mapped into clean sf spatial data frame polygons.

  • Phase 4 — Future Milestone

    Morphometric Parameter Calculations Planned

    Leveraging sf and terra packages to script high-performance zonal statistics across derived catchments. Extracted matrix parameters will isolate mean slope profiles, baseline elevations, cumulative sub-basin areas, and mathematical drainage density indices.

  • Phase 5 — Future Milestone

    Hydrogeological Matrix & Groundwater Analytics Planned

    Generating terrain surface indices including Topographic Wetness Index (TWI) and Stream Power Index (SPI). These arrays will be programmatically overlayed against British Geological Survey aquifer layers and open-source borehole coordinates to flag surface-to-groundwater interaction nodes.

  • Phase 6 — Future Milestone

    Recharge Zoning & Land Use Integration Planned

    Intersecting composition and configuration landscape ecology metrics with underlying geology matrices to extract an exploratory "Potential Recharge Zone" layer. The output text will evaluate how urban surface fragmentation changes water delivery paths and impacts recharge water quality parameters.

  • Phase 7 — Final Deliverable

    Model-Ready Verification & Portfolio Deployment Planned

    Programmatically filtering topological noise and dangling segments to output a clean, topographically valid stream network optimized for export into professional models (HSPF, CASC2D). The project closes with compilation into a standardized spatial dataframe and public deployment via a structured GitHub repository.
