Current Focus

Project: Integrated Catchment Characterisation for Groundwater Vulnerability (R + QGIS)

Building a reproducible R-based workflow that automates watershed delineation and stream network extraction from raw DEM data across Cheshire and Greater Manchester, then extends into groundwater vulnerability and recharge potential using BGS geology data. Currently in the scoping and setup phase — sourcing DEM tiles and establishing a consistent coordinate reference system before any automation work begins.

Timeline

  • July 2026 — In Progress

    Data Sourcing & Terrain Pre-Processing Active Focus

    Sourcing DEM tiles for the Cheshire / Greater Manchester study area and reprojecting to a consistent metric CRS (EPSG:27700). Prototyping r.watershed flow accumulation thresholds on a single test basin in QGIS before scripting the loop.

  • Next

    Automation via qgisprocess Planned

    Build an R loop using the qgisprocess package to run r.watershed across the full folder of DEMs, outputting basin and stream layers for each sub-catchment.

  • Next

    Morphometric & Hydrogeological Statistics Planned

    Calculate zonal statistics, drainage density, TWI/SPI per sub-basin using sf and terra, then overlay with BGS geology and borehole data to flag likely groundwater– surface water interaction zones.
