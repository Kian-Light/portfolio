Current Focus

Project: Integrated Catchment Characterisation for Groundwater Vulnerability (R + QGIS)

Building a reproducible workflow for automated watershed delineation and stream network extraction using Digital Elevation Model (DEM) data across Cheshire and Greater Manchester. Built in R with integration of QGIS and GRASS GIS tools, the workflow is designed to efficiently process large geospatial datasets and generate model ready hydrological outputs. To characterise hydrological processes and compare conditions across multiple sub-basins, the project calculates key catchment characteristics such as basin area, slope, elevation, drainage density, and Topographic Wetness Index (TWI). Geological and aquifer datasets are incorporated to investigate groundwater vulnerability and recharge potential. The resulting outputs aim to include standardised spatial datasets, catchment statistics, and a cleaned stream network suitable for hydrological modelling.
